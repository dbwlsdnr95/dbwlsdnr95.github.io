<h2>자바 코드 구조 이해</h2>
<pre><code>public class HelloJava {
    public static void main(String[] args) {
        System.out.println("hello java");
    }
}
</code></pre>
<h3>주요 구성</h3>
<p>| 구문                                     | 설명            |
| ---------------------------------------- | --------------- |
| <code>public class HelloJava</code>                 | 클래스 선언     |
| <code>public static void main(String[] args)</code> | 프로그램 시작점 |
| <code>System.out.println()</code>                   | 콘솔 출력 명령  |
| <code>;</code>                                      | 문장 끝 구분자  |</p>
<p><strong>실행 과정</strong></p>
<ol>
<li>프로그램 시작</li>
<li><code>main()</code> 실행</li>
<li><code>"hello java"</code> 출력</li>
<li>종료</li>
</ol>
<hr>
<h2>주석(Comment)</h2>
<ul>
<li><strong>한 줄 주석</strong>: <code>//</code></li>
<li><strong>여러 줄 주석</strong>: <code>/* ... */</code></li>
</ul>
<pre><code>public class CommentJava {
    public static void main(String[] args) {
        // 한 줄 주석
        /* 여러 줄 주석 */
        System.out.println("hello java");
    }
}
</code></pre>
<blockquote>
<p>주석은 프로그램에 영향을 주지 않으며, 사람이 읽기 위한 설명용이다1. Hello World.</p>
</blockquote>
<h2>자바란?</h2>
<p>자바는 <strong>표준 스펙</strong>과 <strong>구현체</strong>로 구성됨.</p>
<p>| 표준 스펙          | 구현체 예시     |
| ------------------ | --------------- |
| 자바 컴파일러, JVM | Oracle OpenJDK  |
|                    | Eclipse Temurin |
|                    | Amazon Corretto |</p>
<p>→ 모두 <strong>서로 호환 가능</strong>
→ OpenJDK에서 Temurin으로 바꿔도 대부분 문제 없이 동작</p>
<hr>
<h2>컴파일과 실행 과정</h2>
<ol>
<li><code>.java</code> (소스 코드) → <code>javac</code> 컴파일러 → <code>.class</code> (바이트코드)</li>
<li><code>.class</code> 실행 시 → JVM(Java Virtual Machine)이 작동</li>
</ol>
<pre><code>javac Hello.java
java Hello
</code></pre>
<blockquote>
<p>IntelliJ에서는 이 과정이 자동으로 처리됨.</p>
</blockquote>
<h2>자바와 OS 독립성</h2>
<p>| 일반 프로그램           | 자바 프로그램                |
| ----------------------- | ---------------------------- |
| OS 전용 실행파일 (.exe) | JVM 위에서 실행되는 <code>.class</code> |
| OS 간 호환 불가         | 모든 OS에서 실행 가능        |</p>
<p>즉,
Windows에서 작성한 Java 코드 → Mac/Linux에서도 실행 가능.
운영체제 호환 문제를 <strong>JVM이 해결</strong>해 준다1. Hello World.</p>
