9:T12c1,<h1>연산자 (Operators)</h1>
<p>자바에서 <strong>연산자(operator)</strong> 는 값을 계산·비교·대입하기 위한 기호이다.
예: <code>+</code>, <code>-</code>, <code>*</code>, <code>/</code>, <code>==</code>, <code>&#x26;&#x26;</code>, <code>=</code> 등 3. 연산자</p>
<hr>
<h2>1. 연산자와 피연산자</h2>
<pre><code>3 + 4
a + b
</code></pre>
<ul>
<li><strong>연산자 (Operator)</strong>: 연산 기호 (<code>+</code>, <code>-</code> 등)</li>
<li><strong>피연산자 (Operand)</strong>: 연산 대상 (<code>3</code>, <code>4</code>, <code>a</code>, <code>b</code>)</li>
</ul>
<hr>
<h2>2. 산술 연산자</h2>
<p>숫자 계산에 사용되는 기본 연산자 3. 연산자</p>
<p>| 연산자 | 의미   |
| ------ | ------ |
| <code>+</code>    | 더하기 |
| <code>-</code>    | 빼기   |
| <code>*</code>    | 곱하기 |
| <code>/</code>    | 나누기 |
| <code>%</code>    | 나머지 |</p>
<h3>️ 주의 사항</h3>
<ul>
<li>
<p><strong>int / int → int 결과</strong></p>
<pre><code>5 / 2 = 2   // 소수점 버림
</code></pre>
</li>
<li>
<p><strong>0으로 나누기 불가</strong></p>
<pre><code>10 / 0 → ArithmeticException
</code></pre>
</li>
</ul>
<hr>
<h2>3. 문자열 더하기 (<code>+</code>)</h2>
<p>자바에서는 문자열에도 <code>+</code> 연산자를 사용할 수 있다 3. 연산자</p>
<pre><code>"hello" + "world"     // "helloworld"
"a + b = " + 10       // "a + b = 10"
</code></pre>
<h3>핵심 규칙</h3>
<ul>
<li>문자열 + 숫자 → <strong>숫자가 문자열로 변환</strong></li>
<li>문자열이 하나라도 있으면 결과는 <strong>문자열</strong></li>
</ul>
<hr>
<h2>4. 연산자 우선순위</h2>
<p>자바도 수학과 동일한 우선순위를 따른다 3. 연산자</p>
<pre><code>1 + 2 * 3     // 7
(1 + 2) * 3   // 9
</code></pre>
<h3>우선순위 핵심 정리</h3>
<ol>
<li><code>()</code></li>
<li>단항 연산자 (<code>++</code>, <code>--</code>, <code>!</code>)</li>
<li>산술 (<code>* / %</code> → <code>+ -</code>)</li>
<li>비교 (<code>&#x3C;</code>, <code>></code>, <code>&#x3C;=</code>, <code>>=</code>)</li>
<li>논리 (<code>&#x26;&#x26;</code>, <code>||</code>)</li>
<li>대입 (<code>=</code>)</li>
</ol>
<p><strong>애매하면 무조건 괄호 사용</strong></p>
<hr>
<h2>5. 증감 연산자 (<code>++</code>, <code>--</code>)</h2>
<p>값을 <strong>1 증가 / 감소</strong>시킨다 3. 연산자</p>
<pre><code>++a;  // a = a + 1
--a;  // a = a - 1
</code></pre>
<h3>전위 / 후위 차이</h3>
<p>| 형태  | 설명              |
| ----- | ----------------- |
| <code>++a</code> | 먼저 증가 후 사용 |
| <code>a++</code> | 먼저 사용 후 증가 |</p>
<pre><code>b = ++a; // a 증가 → b 대입
b = a++; // b 대입 → a 증가
</code></pre>
<hr>
<h2>6. 비교 연산자</h2>
<p>두 값을 비교하여 <strong>boolean(true/false)</strong> 반환 3. 연산자</p>
<p>| 연산자 | 의미        |
| ------ | ----------- |
| <code>==</code>   | 같다        |
| <code>!=</code>   | 다르다      |
| <code>></code>    | 크다        |
| <code>&#x3C;</code>    | 작다        |
| <code>>=</code>   | 크거나 같다 |
| <code>&#x3C;=</code>   | 작거나 같다 |</p>
<h3>️ 문자열 비교 주의</h3>
<pre><code>str1 == str2      //  사용 금지
str1.equals(str2) //  문자열 비교
</code></pre>
<hr>
<h2>7. 논리 연산자</h2>
<p>boolean 값의 논리 연산 3. 연산자</p>
<p>| 연산자 | 의미            |
| ------ | --------------- |
| <code>&#x26;&#x26;</code>   | AND (모두 true) |
| <code>     |                 | |</code>!`    | NOT (반전)      |</p>
<pre><code>a > 10 &#x26;&#x26; a &#x3C; 20
</code></pre>
<hr>
<h2>8. 대입 연산자</h2>
<p>값을 변수에 저장 3. 연산자</p>
<pre><code>int a = 5;
</code></pre>
<h3>복합 대입 연산자</h3>
<pre><code>a += 3;  // a = a + 3
a *= 2;  // a = a * 2
</code></pre>
<hr>
<h2>9. 실습 문제 핵심 요약</h2>
<h3>평균 계산 (int)</h3>
<pre><code>int avg = sum / 3; // 소수점 버림
</code></pre>
<h3>평균 계산 (double)</h3>
<pre><code>double avg = sum / 3;
</code></pre>
<h3>범위 체크</h3>
<pre><code>score >= 80 &#x26;&#x26; score &#x3C;= 100
</code></pre>
<hr>
<h2>최종 정리</h2>
<h3>자주 사용하는 연산자</h3>
<ul>
<li>산술: <code>+ - * / %</code></li>
<li>증감: <code>++ --</code></li>
<li>비교: <code>== != > &#x3C; >= &#x3C;=</code></li>
<li>논리: <code>&#x26;&#x26; || !</code></li>
<li>대입: <code>= += -= *= /= %=</code></li>
</ul>
<h3>핵심 원칙</h3>
<ul>
<li><strong>연산자 우선순위는 상식선</strong></li>
<li><strong>애매하면 괄호</strong></li>
<li><strong>문자열 비교는 equals()</strong></li>
<li><strong>가독성이 유지보수보다 중요</strong></li>
</ul>
