9:T1843,<h1>조건문 (Conditional Statements)</h1>
<p>조건문은 <strong>특정 조건에 따라 서로 다른 코드를 실행</strong>하기 위해 사용한다.
자바의 대표적인 조건문은 <code>if</code>, <code>switch</code>, <code>삼항 연산자</code>가 있다. 4. 조건문</p>
<hr>
<h2>1. if 문</h2>
<h3>1.1 if 기본 구조</h3>
<pre><code>if (condition) {
    // 조건이 참(true)일 때 실행
}
</code></pre>
<ul>
<li>조건식은 <code>true</code> 또는 <code>false</code>로 평가됨</li>
<li><code>{}</code> 안의 코드를 <strong>코드 블록</strong>이라 함</li>
</ul>
<hr>
<h3>1.2 if 단독 사용 예제</h3>
<pre><code>if (age >= 18) {
    System.out.println("성인입니다.");
}
</code></pre>
<ul>
<li>조건이 참일 경우에만 실행</li>
<li>거짓이면 아무 동작도 하지 않음</li>
</ul>
<hr>
<h2>2. if - else 문</h2>
<h3>2.1 기본 구조</h3>
<pre><code>if (condition) {
    // 조건이 참일 때
} else {
    // 조건이 거짓일 때
}
</code></pre>
<ul>
<li>두 경우 중 <strong>하나만 실행</strong></li>
<li>서로 배타적인 조건에 사용</li>
</ul>
<h3>예제</h3>
<pre><code>if (age >= 18) {
    System.out.println("성인입니다.");
} else {
    System.out.println("미성년자입니다.");
}
</code></pre>
<hr>
<h2>3. else if 문</h2>
<h3>3.1 필요성</h3>
<p>여러 조건을 <strong>순서대로 비교</strong>해야 할 때 사용
앞의 조건이 참이면 뒤 조건은 <strong>검사하지 않음</strong></p>
<h3>3.2 기본 구조</h3>
<pre><code>if (condition1) {
    // 실행
} else if (condition2) {
    // 실행
} else {
    // 모두 거짓일 때
}
</code></pre>
<hr>
<h3>3.3 예제 (연령 분류)</h3>
<pre><code>if (age &#x3C;= 7) {
    System.out.println("미취학");
} else if (age &#x3C;= 13) {
    System.out.println("초등학생");
} else if (age &#x3C;= 16) {
    System.out.println("중학생");
} else if (age &#x3C;= 19) {
    System.out.println("고등학생");
} else {
    System.out.println("성인");
}
</code></pre>
<h4>특징</h4>
<ul>
<li><strong>조건은 위에서 아래로 평가</strong></li>
<li><strong>딱 하나의 블록만 실행</strong></li>
</ul>
<hr>
<h2>4. if vs else if vs 여러 if</h2>
<h3>4.1 else if 사용 (서로 연관된 조건)</h3>
<pre><code>if (condition1) {
} else if (condition2) {
}
</code></pre>
<ul>
<li>하나만 실행됨</li>
</ul>
<h3>4.2 여러 if 사용 (독립 조건)</h3>
<pre><code>if (condition1) {
}
if (condition2) {
}
</code></pre>
<ul>
<li>조건이 맞으면 <strong>모두 실행 가능</strong></li>
</ul>
<hr>
<h3>4.3 할인 시스템 예제 (독립 조건)</h3>
<pre><code>if (price >= 10000) {
    discount += 1000;
}
if (age &#x3C;= 10) {
    discount += 1000;
}
</code></pre>
<ul>
<li>여러 할인 <strong>동시 적용 가능</strong></li>
<li>이 경우 <code>else if</code> 사용 x</li>
</ul>
<hr>
<h2>5. if 문 중괄호 <code>{}</code> 생략</h2>
<h3>5.1 생략 가능 조건</h3>
<pre><code>if (true)
    System.out.println("실행됨");
</code></pre>
<h3>5.2 주의점 (버그 유발)</h3>
<pre><code>if (true)
    System.out.println("실행됨");
    System.out.println("항상 실행됨");
</code></pre>
<h3>권장 스타일</h3>
<pre><code>if (true) {
    System.out.println("실행됨");
    System.out.println("함께 실행");
}
</code></pre>
<ul>
<li>가독성 ↑</li>
<li>유지보수성 ↑</li>
</ul>
<hr>
<h2>6. switch 문</h2>
<h3>6.1 switch 기본 구조</h3>
<pre><code>switch (value) {
    case 1:
        // 실행
        break;
    default:
        // 나머지
}
</code></pre>
<ul>
<li><strong>값의 일치 여부만 비교</strong></li>
<li>조건식(<code>></code>, <code>&#x3C;</code>) 사용 불가</li>
</ul>
<hr>
<h3>6.2 switch 예제</h3>
<pre><code>switch (grade) {
    case 1:
        coupon = 1000;
        break;
    case 2:
        coupon = 2000;
        break;
    case 3:
        coupon = 3000;
        break;
    default:
        coupon = 500;
}
</code></pre>
<hr>
<h3>6.3 break 없는 switch (fall-through)</h3>
<pre><code>case 2:
case 3:
    coupon = 3000;
    break;
</code></pre>
<ul>
<li>여러 case를 하나로 묶을 수 있음</li>
</ul>
<hr>
<h2>7. if 문 vs switch 문</h2>
<p>| 구분   | if 문            | switch 문           |
| ------ | ---------------- | ------------------- |
| 조건   | 논리식 가능      | 값 비교만           |
| 범위   | 넓음             | 제한적              |
| 가독성 | 복잡해질 수 있음 | 특정 값 분기에 적합 |</p>
<hr>
<h2>8. Java 14 새로운 switch 문</h2>
<pre><code>int coupon = switch (grade) {
    case 1 -> 1000;
    case 2 -> 2000;
    case 3 -> 3000;
    default -> 500;
};
</code></pre>
<h3>특징</h3>
<ul>
<li><code>-></code> 사용</li>
<li>값 반환 가능</li>
<li>코드 간결</li>
</ul>
<hr>
<h2>9. 삼항 연산자 (조건 연산자)</h2>
<h3>9.1 기본 형태</h3>
<pre><code>(조건) ? 참_표현식 : 거짓_표현식
</code></pre>
<h3>예제</h3>
<pre><code>String status = (age >= 18) ? "성인" : "미성년자";
</code></pre>
<h3>특징</h3>
<ul>
<li>if-else를 한 줄로 표현</li>
<li><strong>표현식만 가능</strong> (코드 블록 x)</li>
</ul>
<hr>
<h2>10. 문제 풀이 핵심 정리</h2>
<h3>10.1 학점 계산</h3>
<pre><code>if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("F");
}
</code></pre>
<hr>
<h3>10.2 거리별 이동 수단</h3>
<pre><code>if (distance &#x3C;= 1) {
    System.out.println("도보");
} else if (distance &#x3C;= 10) {
    System.out.println("자전거");
}
</code></pre>
<hr>
<h3>10.3 삼항 연산자 활용</h3>
<pre><code>int max = (a > b) ? a : b;
String result = (x % 2 == 0) ? "짝수" : "홀수";
</code></pre>
<hr>
<h2>최종 정리</h2>
<ul>
<li><strong>if / else if</strong> → 조건 분기</li>
<li><strong>여러 if</strong> → 독립 조건</li>
<li><strong>switch</strong> → 특정 값 분기</li>
<li><strong>삼항 연산자</strong> → 단순 조건 처리</li>
<li><strong>중괄호는 항상 사용하는 것이 안전</strong></li>
</ul>
