a:T167b,<h1>변수 (Variable)</h1>
<h2>개요</h2>
<ul>
<li>변수(variable)는 <strong>값을 저장하고 재사용하기 위한 데이터 저장소</strong></li>
<li>값이 변할 수 있기 때문에 <em>변수</em>라고 부른다</li>
</ul>
<hr>
<h2>1. 변수의 필요성</h2>
<h3>문제 상황</h3>
<pre><code>System.out.println(10);
System.out.println(10);
System.out.println(10);
</code></pre>
<ul>
<li>같은 값을 여러 번 사용하면
<ul>
<li>값 변경 시 <strong>모든 코드를 수정해야 함</strong></li>
<li>유지보수에 불리함</li>
</ul>
</li>
</ul>
<h3>해결: 변수 사용</h3>
<pre><code>int a = 10;
System.out.println(a);
System.out.println(a);
System.out.println(a);
</code></pre>
<ul>
<li>변수 <code>a</code> 값만 변경하면 모든 출력이 함께 변경됨</li>
</ul>
<hr>
<h2>2. 변수 선언과 초기화</h2>
<h3>변수 선언</h3>
<pre><code>int a;
</code></pre>
<ul>
<li>정수(int)를 저장할 수 있는 공간을 메모리에 생성</li>
</ul>
<h3>변수 초기화</h3>
<pre><code>a = 10;
</code></pre>
<ul>
<li>선언된 변수에 <strong>처음으로 값 저장</strong></li>
</ul>
<h3>선언 + 초기화 동시에</h3>
<pre><code>int a = 10;
</code></pre>
<h3>여러 변수 선언</h3>
<pre><code>int a, b;
int c = 3, d = 4;
</code></pre>
<hr>
<h2>3. 변수 값 읽기 &#x26; 변경</h2>
<h3>값 읽기</h3>
<pre><code>System.out.println(a);
</code></pre>
<ul>
<li>변수 이름을 사용하면 저장된 값이 읽힘</li>
<li>여러 번 읽어도 값은 사라지지 않음</li>
</ul>
<h3>값 변경</h3>
<pre><code>a = 10;
a = 50;
</code></pre>
<ul>
<li>기존 값은 <strong>완전히 덮어쓰기</strong></li>
<li>프로그램은 <strong>위에서 아래로 순차 실행</strong></li>
</ul>
<hr>
<h2>4. 변수는 반드시 초기화해야 한다</h2>
<p>잘못된 코드</p>
<pre><code>int a;
System.out.println(a);
</code></pre>
<p>컴파일 에러</p>
<pre><code>variable a might not have been initialized
</code></pre>
<h3>이유</h3>
<ul>
<li>메모리에는 쓰레기 값이 존재할 수 있음</li>
<li>자바는 이를 방지하기 위해 <strong>지역 변수 초기화 강제</strong></li>
</ul>
<blockquote>
<p>클래스 변수 / 인스턴스 변수는 자동 초기화됨 (추후 학습)</p>
</blockquote>
<hr>
<h2>5. 변수 타입(Type) – 기본</h2>
<pre><code>int a = 100;          // 정수
double b = 10.5;     // 실수
boolean c = true;    // 참/거짓
char d = 'A';        // 문자 1개
String e = "Hello";  // 문자열
</code></pre>
<h3>주요 타입 정리</h3>
<p>| 타입    | 설명                 |
| ------- | -------------------- |
| int     | 정수                 |
| double  | 실수                 |
| boolean | true / false         |
| char    | 문자 1개             |
| String  | 문자열 (특별한 타입) |</p>
<hr>
<h2>6. 리터럴(Literal)</h2>
<ul>
<li>코드에 <strong>직접 작성한 고정 값</strong></li>
</ul>
<pre><code>int a = 100;
double b = 10.5;
boolean c = true;
</code></pre>
<ul>
<li>변수 값은 변하지만 <strong>리터럴은 변하지 않음</strong></li>
</ul>
<hr>
<h2>7. 숫자 타입 상세</h2>
<h3>정수형</h3>
<p>| 타입  | 범위             | 크기 |
| ----- | ---------------- | ---- |
| byte  | -128 ~ 127       | 1B   |
| short | -32,768 ~ 32,767 | 2B   |
| int   | 약 ±20억         | 4B   |
| long  | 매우 큰 정수     | 8B   |</p>
<pre><code>long l = 10000000000L;
</code></pre>
<h3>실수형</h3>
<p>| 타입   | 정밀도 | 크기 |
| ------ | ------ | ---- |
| float  | 7자리  | 4B   |
| double | 15자리 | 8B   |</p>
<pre><code>float f = 10.0f;
double d = 10.0;
</code></pre>
<hr>
<h2>8. 실무에서 자주 쓰는 타입</h2>
<ul>
<li><strong>정수</strong>: <code>int</code>, <code>long</code></li>
<li><strong>실수</strong>: <code>double</code></li>
<li><strong>논리</strong>: <code>boolean</code></li>
<li><strong>문자열</strong>: <code>String</code></li>
</ul>
<p>거의 사용 안 함</p>
<ul>
<li><code>byte</code>, <code>short</code>, <code>float</code>, <code>char</code></li>
</ul>
<blockquote>
<p>메모리보다 <strong>가독성과 개발 속도</strong>가 더 중요</p>
</blockquote>
<hr>
<h2>9. 변수 명명 규칙</h2>
<h3>필수 규칙 (안 지키면 컴파일 에러)</h3>
<ul>
<li>숫자로 시작</li>
<li>공백</li>
<li>예약어 사용</li>
<li>사용 가능 문자: 영문, 숫자, <code>_</code>, <code>$</code></li>
</ul>
<h3>관례 (사실상 필수)</h3>
<ul>
<li><strong>소문자 시작</strong></li>
<li><strong>camelCase 사용</strong></li>
</ul>
<pre><code>int studentCount;
int orderDetail;
</code></pre>
<h3>자바 전체 네이밍 규칙 요약</h3>
<p>| 대상        | 규칙              |
| ----------- | ----------------- |
| 클래스      | 대문자 시작       |
| 변수/메서드 | 소문자 시작       |
| 상수        | 전부 대문자 + <code>_</code> |
| 패키지      | 전부 소문자       |</p>
<hr>
<h2>10. 연습 문제 요약</h2>
<h3>문제 1 – 변수로 중복 제거</h3>
<pre><code>int num1 = 4;
int num2 = 3;
System.out.println(num1 + num2);
</code></pre>
<h3>문제 2 – 합 계산</h3>
<pre><code>int num1 = 10;
int num2 = 20;
int sum = num1 + num2;
System.out.println(sum);
</code></pre>
<h3>문제 3 – long / boolean</h3>
<pre><code>long value = 10000000000L;
boolean flag = true;
</code></pre>
<hr>
<h2>핵심 정리</h2>
<ul>
<li>변수 = <strong>값을 저장하는 메모리 공간</strong></li>
<li>사용 전 <strong>반드시 초기화</strong></li>
<li>타입에 맞는 값만 저장 가능</li>
<li>실무에서는 <strong>int / double / boolean / String</strong> 위주 사용</li>
<li>변수 이름은 <strong>의미 있게</strong></li>
</ul>
