변수 (Variable)

개요

  • 변수(variable)는 값을 저장하고 재사용하기 위한 데이터 저장소
  • 값이 변할 수 있기 때문에 변수라고 부른다

1. 변수의 필요성

문제 상황

System.out.println(10);
System.out.println(10);
System.out.println(10);
  • 같은 값을 여러 번 사용하면
    • 값 변경 시 모든 코드를 수정해야 함
    • 유지보수에 불리함

해결: 변수 사용

int a = 10;
System.out.println(a);
System.out.println(a);
System.out.println(a);
  • 변수 a 값만 변경하면 모든 출력이 함께 변경됨

2. 변수 선언과 초기화

변수 선언

int a;
  • 정수(int)를 저장할 수 있는 공간을 메모리에 생성

변수 초기화

a = 10;
  • 선언된 변수에 처음으로 값 저장

선언 + 초기화 동시에

int a = 10;

여러 변수 선언

int a, b;
int c = 3, d = 4;

3. 변수 값 읽기 & 변경

값 읽기

System.out.println(a);
  • 변수 이름을 사용하면 저장된 값이 읽힘
  • 여러 번 읽어도 값은 사라지지 않음

값 변경

a = 10;
a = 50;
  • 기존 값은 완전히 덮어쓰기
  • 프로그램은 위에서 아래로 순차 실행

4. 변수는 반드시 초기화해야 한다

잘못된 코드

int a;
System.out.println(a);

컴파일 에러

variable a might not have been initialized

이유

  • 메모리에는 쓰레기 값이 존재할 수 있음
  • 자바는 이를 방지하기 위해 지역 변수 초기화 강제

클래스 변수 / 인스턴스 변수는 자동 초기화됨 (추후 학습)

5. 변수 타입(Type) – 기본

int a = 100;          // 정수
double b = 10.5;     // 실수
boolean c = true;    // 참/거짓
char d = 'A';        // 문자 1개
String e = "Hello";  // 문자열

주요 타입 정리

| 타입 | 설명 | | ------- | -------------------- | | int | 정수 | | double | 실수 | | boolean | true / false | | char | 문자 1개 | | String | 문자열 (특별한 타입) |

6. 리터럴(Literal)

  • 코드에 직접 작성한 고정 값
int a = 100;
double b = 10.5;
boolean c = true;
  • 변수 값은 변하지만 리터럴은 변하지 않음

7. 숫자 타입 상세

정수형

| 타입 | 범위 | 크기 | | ----- | ---------------- | ---- | | byte | -128 ~ 127 | 1B | | short | -32,768 ~ 32,767 | 2B | | int | 약 ±20억 | 4B | | long | 매우 큰 정수 | 8B |

long l = 10000000000L;

실수형

| 타입 | 정밀도 | 크기 | | ------ | ------ | ---- | | float | 7자리 | 4B | | double | 15자리 | 8B |

float f = 10.0f;
double d = 10.0;

8. 실무에서 자주 쓰는 타입

  • 정수: int, long
  • 실수: double
  • 논리: boolean
  • 문자열: String

거의 사용 안 함

  • byte, short, float, char

메모리보다 가독성과 개발 속도가 더 중요

9. 변수 명명 규칙

필수 규칙 (안 지키면 컴파일 에러)

  • 숫자로 시작
  • 공백
  • 예약어 사용
  • 사용 가능 문자: 영문, 숫자, _, $

관례 (사실상 필수)

  • 소문자 시작
  • camelCase 사용
int studentCount;
int orderDetail;

자바 전체 네이밍 규칙 요약

| 대상 | 규칙 | | ----------- | ----------------- | | 클래스 | 대문자 시작 | | 변수/메서드 | 소문자 시작 | | 상수 | 전부 대문자 + _ | | 패키지 | 전부 소문자 |

10. 연습 문제 요약

문제 1 – 변수로 중복 제거

int num1 = 4;
int num2 = 3;
System.out.println(num1 + num2);

문제 2 – 합 계산

int num1 = 10;
int num2 = 20;
int sum = num1 + num2;
System.out.println(sum);

문제 3 – long / boolean

long value = 10000000000L;
boolean flag = true;

핵심 정리

  • 변수 = 값을 저장하는 메모리 공간
  • 사용 전 반드시 초기화
  • 타입에 맞는 값만 저장 가능
  • 실무에서는 int / double / boolean / String 위주 사용
  • 변수 이름은 의미 있게
