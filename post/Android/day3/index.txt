a:T1a99,<h1>Android 기본 컴포넌트 및 보안 분석 정리</h1>
<h2>Android Component</h2>
<p>| 컴포넌트              | 역할                       | 예시                                                 |
| --------------------- | -------------------------- | ---------------------------------------------------- |
| <strong>Activity</strong>          | 사용자 화면(UI) 구성       | 로그인 액티비티, 친구 목록 액티비티, 대화창 액티비티 |
| <strong>Service</strong>           | 백그라운드 실행 기능       | 음악 재생, 파일 다운로드 등                          |
| <strong>ContentProvider</strong>   | 저장된 데이터 공유 및 전달 | 주소록, 메모, 앱 간 데이터 공유                      |
| <strong>BroadcastReceiver</strong> | 이벤트 감지 및 처리        | 배터리 상태, 네트워크 변화 등                        |</p>
<hr>
<h2>exported 속성</h2>
<ul>
<li><code>android:exported="false"</code> → 외부 앱에서 접근 불가</li>
<li><code>android:exported="true"</code> → 다른 앱에서도 호출 가능 (주의 필요)</li>
</ul>
<p>예시:</p>
<pre><code>&#x3C;activity
    android:name=".MainActivity"
    android:exported="false" />
</code></pre>
<hr>
<h2>️ 예시 앱 구조</h2>
<p>| 앱           | 주요 기능              |
| ------------ | ---------------------- |
| <strong>전화앱</strong>   | 통화, 연락처 접근      |
| <strong>카카오톡</strong> | 메신저, 친구목록, 대화 |</p>
<hr>
<h2>Logcat (로그 출력)</h2>
<ul>
<li>
<p><strong>로그 확인</strong>
Android Studio → <code>Logcat</code> 창
또는 ADB 명령어:</p>
<pre><code>adb logcat
</code></pre>
</li>
<li>
<p><strong>출력 예시</strong></p>
<pre><code>I/MainActivity: 로그인 성공
E/Database: Connection failed
</code></pre>
</li>
</ul>
<hr>
<h2>️ 데이터 저장소 구조</h2>
<p>| 저장 방식             | 경로                                  | 설명                                    |
| --------------------- | ------------------------------------- | --------------------------------------- |
| <strong>SharedPreferences</strong> | <code>/data/data/&#x3C;패키지명>/shared_prefs/</code> | 간단한 key-value 저장 (예: 로그인 정보) |
| <strong>SQLite Database</strong>   | <code>/data/data/&#x3C;패키지명>/databases/</code>    | 구조화된 데이터 저장                    |
| <strong>Sandbox</strong>           | 앱별 독립 저장 공간                   | 앱 간 접근 제한                         |</p>
<hr>
<h2>️ SharedPreferences 예시</h2>
<ul>
<li>
<p>저장:</p>
<pre><code>SharedPreferences pref = getSharedPreferences("userInfo", MODE_PRIVATE);
pref.edit().putString("id", "user123").apply();
</code></pre>
</li>
<li>
<p>파일 예시 (<code>userInfo.xml</code>):</p>
<pre><code>&#x3C;map>
    &#x3C;string name="id">user123&#x3C;/string>
&#x3C;/map>
</code></pre>
</li>
</ul>
<hr>
<h2>SQLite Database 분석</h2>
<ul>
<li>
<p>경로:</p>
<pre><code>/data/data/&#x3C;패키지명>/databases/
</code></pre>
</li>
<li>
<p>접근:</p>
<pre><code>adb shell
cd /data/data/&#x3C;패키지명>/databases/
sqlite3 test.db
.tables
SELECT * FROM user;
</code></pre>
</li>
</ul>
<hr>
<h2>메모리 내 중요 정보 분석</h2>
<p>| 명령어                          | 설명                        |
| ------------------------------- | --------------------------- |
| <code>am dumpheap &#x3C;pid> &#x3C;경로></code>      | 앱 프로세스 메모리 덤프     |
| <code>strings test.hprof > test.txt</code> | 메모리 내 문자열 추출       |
| <code>adb shell ps</code>                  | 현재 실행중인 프로세스 확인 |</p>
<p><strong>예시</strong></p>
<pre><code>adb shell ps | grep aos.app017
adb shell am dumpheap 29429 /data/local/tmp/test.hprof
adb shell chmod 777 /data/local/tmp/test.hprof
adb shell strings /data/local/tmp/test.hprof > /sdcard/test.txt
</code></pre>
<hr>
<h2>Activity 강제 실행 (Activity Manager)</h2>
<pre><code>adb shell am start 패키지명/.액티비티이름
</code></pre>
<h3>Extra 전달 예시</h3>
<p>| 타입    | 옵션   | 예시                    |
| ------- | ------ | ----------------------- |
| Boolean | <code>--ez</code> | <code>--ez check true</code>       |
| String  | <code>--es</code> | <code>--es title "공지사항"</code> |
| Integer | <code>--ei</code> | <code>--ei postid 3</code>         |</p>
<p><strong>예시 명령:</strong></p>
<pre><code>adb shell am start --es title abc --es content asdf --ei postid 3
</code></pre>
