a:T1386,<p>모바일 앱 분석 (정적·동적 분석)</p>
<h2>1. 분석 개요</h2>
<h3>정적 분석</h3>
<ul>
<li>앱을 <strong>실행하지 않은 상태</strong>에서 분석</li>
<li>문자열, 함수, 로직을 <strong>거꾸로 추적</strong>하며 기능 파악</li>
<li>주요 도구
<ul>
<li>jadx</li>
<li>ghidra</li>
</ul>
</li>
<li>Java 코드, Smali 코드 기반 분석</li>
</ul>
<h3>동적 분석</h3>
<ul>
<li>앱을 <strong>실행한 상태</strong>에서 분석</li>
<li>함수 호출 흐름, 리턴값, 변수 확인</li>
<li>런타임 변조 및 후킹</li>
<li>주요 도구
<ul>
<li>Frida</li>
<li>Xposed</li>
<li>Burp와 유사한 방식의 런타임 조작</li>
</ul>
</li>
</ul>
<hr>
<h2>2. 환경 구축</h2>
<h3>필수 구성 요소</h3>
<ul>
<li>Python</li>
<li>adb</li>
<li>기기 내 <code>frida-server</code></li>
<li>PC 내 <code>frida</code>, <code>frida-tools</code></li>
<li>코드 편집기</li>
</ul>
<hr>
<h3>PC Frida 설치</h3>
<pre><code>pip install frida-tools
</code></pre>
<hr>
<h3>모바일 기기 Frida Server 설치</h3>
<ul>
<li>PC에 설치한 Frida 버전과 <strong>동일한 버전</strong> 사용</li>
<li>GitHub Frida Releases에서 다운로드</li>
<li>예시:
<ul>
<li><code>frida-server-17.5.2-android-x86_64.xz</code></li>
</ul>
</li>
</ul>

</code></pre>
<ul>

<h2>3. Frida Tools 정리</h2>
<h3>주요 명령어</h3>
<p>| 명령어           | 설명                          |
| ---------------- | ----------------------------- |
| frida            | CLI 기반 Frida 쉘             |
| frida-ps         | 실행 중인 프로세스 목록       |
| frida-trace      | 함수 호출 자동 추적           |
| frida-kill       | 실행 중인 앱 종료             |
| frida-ls-devices | 연결된 기기 목록              |
| frida-discover   | 메모리 어셈블리 분석 (비사용) |</p>
<hr>
<h3>기본 사용법</h3>
<pre><code>frida -U 앱이름
frida -U -f 패키지명
frida -U -l 공격스크립트.js -f 패키지명
</code></pre>
<ul>
<li><code>-U</code> : USB 디버깅 기기</li>
<li><code>-f</code> : 앱 spawn 실행</li>
</ul>
<hr>
<h3>실행 중 앱 확인</h3>
<pre><code>frida-ps -U
frida-ps -Uai
</code></pre>
<ul>
<li><code>-ai</code> : 설치된 앱 + PID 출력</li>
<li>Frida 서버 동작 여부와는 무관</li>
</ul>
<hr>
<h2>4. frida-trace 상세</h2>
<h3>목적</h3>
<ul>
<li>함수 호출 흐름을 자동으로 추적</li>
<li>정확한 분석 전 <strong>전체 구조 파악용</strong></li>
</ul>
<hr>
<h3>기본 사용법</h3>
<pre><code>frida-trace -U -i '네이티브함수' -j '자바함수' 앱이름
</code></pre>
<p>옵션 설명:</p>
<ul>
<li><code>-i</code> : 네이티브 함수 포함</li>
<li><code>-j</code> : Java 함수 포함</li>
<li><code>-x</code>, <code>-J</code> : 제외할 함수 지정</li>
<li><code>-f</code> : 앱 재실행 (spawn)</li>
</ul>
<hr>
<h3>예제</h3>
<pre><code>frida-trace -U -j "패키지명.MainActivity!*" -f 패키지명
frida-trace -U -j "패키지명.GameView!*" -J "패키지명.GameView!draw" -f 패키지명
</code></pre>
<ul>
<li><code>draw()</code> 함수는 호출이 과도하므로 제외</li>
</ul>
<hr>
<h3>frida-trace 결과 파일</h3>
<ul>
<li>자동 생성 위치:</li>
</ul>
<pre><code>_handlers_/클래스명/메소드.js
</code></pre>
<ul>
<li>생성된 JS 파일은 <strong>frida로 바로 실행 불가</strong></li>
<li>직접 수정 필요</li>
</ul>
