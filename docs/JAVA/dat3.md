# 연산자 (Operators)

자바에서 **연산자(operator)** 는 값을 계산·비교·대입하기 위한 기호이다.
 예: `+`, `-`, `*`, `/`, `==`, `&&`, `=` 등 3. 연산자

------

## 1. 연산자와 피연산자

```
3 + 4
a + b
```

- **연산자 (Operator)**: 연산 기호 (`+`, `-` 등)
- **피연산자 (Operand)**: 연산 대상 (`3`, `4`, `a`, `b`)

------

## 2. 산술 연산자

숫자 계산에 사용되는 기본 연산자 3. 연산자

| 연산자 | 의미   |
| ------ | ------ |
| `+`    | 더하기 |
| `-`    | 빼기   |
| `*`    | 곱하기 |
| `/`    | 나누기 |
| `%`    | 나머지 |

###  주의 사항

- **int / int  int 결과**

  ```
  5 / 2 = 2   // 소수점 버림
  ```

- **0으로 나누기 불가**

  ```
  10 / 0  ArithmeticException
  ```

------

## 3. 문자열 더하기 (`+`)

자바에서는 문자열에도 `+` 연산자를 사용할 수 있다 3. 연산자

```
"hello" + "world"     // "helloworld"
"a + b = " + 10       // "a + b = 10"
```

### 핵심 규칙

- 문자열 + 숫자  **숫자가 문자열로 변환**
- 문자열이 하나라도 있으면 결과는 **문자열**

------

## 4. 연산자 우선순위

자바도 수학과 동일한 우선순위를 따른다 3. 연산자

```
1 + 2 * 3     // 7
(1 + 2) * 3   // 9
```

### 우선순위 핵심 정리

1. `()`
2. 단항 연산자 (`++`, `--`, `!`)
3. 산술 (`* / %`  `+ -`)
4. 비교 (`<`, `>`, `<=`, `>=`)
5. 논리 (`&&`, `||`)
6. 대입 (`=`)

 **애매하면 무조건 괄호 사용**

------

## 5. 증감 연산자 (`++`, `--`)

값을 **1 증가 / 감소**시킨다 3. 연산자

```
++a;  // a = a + 1
--a;  // a = a - 1
```

### 전위 / 후위 차이

| 형태  | 설명              |
| ----- | ----------------- |
| `++a` | 먼저 증가 후 사용 |
| `a++` | 먼저 사용 후 증가 |

```
b = ++a; // a 증가  b 대입
b = a++; // b 대입  a 증가
```

------

## 6. 비교 연산자

두 값을 비교하여 **boolean(true/false)** 반환 3. 연산자

| 연산자 | 의미        |
| ------ | ----------- |
| `==`   | 같다        |
| `!=`   | 다르다      |
| `>`    | 크다        |
| `<`    | 작다        |
| `>=`   | 크거나 같다 |
| `<=`   | 작거나 같다 |

###  문자열 비교 주의

```
str1 == str2      //  사용 금지
str1.equals(str2) //  문자열 비교
```

------

## 7. 논리 연산자

boolean 값의 논리 연산 3. 연산자

| 연산자 | 의미            |
| ------ | --------------- |
| `&&`   | AND (모두 true) |
| `      |                 |
| `!`    | NOT (반전)      |

```
a > 10 && a < 20
```

------

## 8. 대입 연산자

값을 변수에 저장 3. 연산자

```
int a = 5;
```

### 복합 대입 연산자

```
a += 3;  // a = a + 3
a *= 2;  // a = a * 2
```

------

## 9. 실습 문제 핵심 요약

### 평균 계산 (int)

```
int avg = sum / 3; // 소수점 버림
```

### 평균 계산 (double)

```
double avg = sum / 3;
```

### 범위 체크

```
score >= 80 && score <= 100
```

------

##  최종 정리

### 자주 사용하는 연산자

- 산술: `+ - * / %`
- 증감: `++ --`
- 비교: `== != > < >= <=`
- 논리: `&& || !`
- 대입: `= += -= *= /= %=`

### 핵심 원칙

- **연산자 우선순위는 상식선**
- **애매하면 괄호**
- **문자열 비교는 equals()**
- **가독성이 유지보수보다 중요**
